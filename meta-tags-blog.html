The <meta> tags

Definition

The <meta> tag is a source code that is not displayed on the page but is machine parsable and contains data defining the web page’s data to search engines.

<meta> tag is always placed inside the <head> element.

Typical attributes present in <meta> element are character-set, name, content, and http-equiv.

Attributes

  1. charset: The character encoding used by the HTML document.

    Ex: <meta charset = “utf-8”> : utf-8 is an universal character set that contains almost all characters from every human language.

  2. name: Name of the metadata;

    content: Value associated with name attribute.

    Ex: <meta name=“author” content=“Name of the author”>
    <meta name = “description” content = “Description of the content in the web page”>

  3. http-equiv: HTTP header for information of the content attribute.

    Ex: <meta http-equiv = ”expires” content = “14 March 2022 11:44:00 IST”>
    In the above example, when the date is reached, the page will be reloaded even when the page is in the cache.
